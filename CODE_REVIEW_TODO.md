  - **Problem:** EF Core's SQLite-Provider unterstützt keine echten rowversion/Timestamp-Spalten. Grep bestätigt: kein CREATE TRIGGER, kein HasDefaultValueSql/randomblob, kein eigener ValueGenerator irgendwo. Beim INSERT setzt EF die Spalte nicht, beim UPDATE bleibt der Wert unverändert.
