# 📚 Book Logger App

Eine Android-App zum Verwalten und Protokollieren deiner Bücher.  
Gebaut mit **.NET MAUI Blazor Hybrid** und **SQLite** als Datenbank.

---

## 🚀 Features (MVP)
- Bücher hinzufügen, bearbeiten, löschen
- Lesefortschritt speichern
- Bewertungen & Notizen zu Büchern
- Regale / Bookshelf-Ansicht
- Statistiken zum Leseverhalten

---

## 🔧 Tech Stack
- [.NET 9 MAUI Blazor Hybrid](https://learn.microsoft.com/dotnet/maui)
- SQLite für lokale Datenspeicherung
- MVVM + Dependency Injection
- GitHub Actions für CI/CD

---

## 📂 Projektstruktur
```

BookLoggerApp/          → MAUI Hauptprojekt
BookLoggerApp.Core/     → Models & Services
BookLoggerApp.Tests/    → Unit Tests

```
